B-MJ

Block-based MiniJava

Block MiniJava is a TypeScript, npm, webpack, and Blockly-based web IDE for constructing MiniJava programs with grammar-aware blocks.

The project saves workspaces as JSON payloads using the .bml extension.

The custom bmj-thrasos renderer uses distinct connector shapes to visually differentiate MiniJava non-terminals and reduce grammatical production errors.
